# Vancouver Housing Prices (2004-2024) Overview

In this project, we will examine Vancouver housing prices for the 20-year period between 2004-2024. This is a topic that is quite personal to me, as someone born and raised in Vancouver.  

This dataset is from [Kaggle](https://www.kaggle.com/datasets/jennyzzhu/vancouver-house-prices-for-past-20-years).
